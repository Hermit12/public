The while loop:

The while loop repeats a block of code as long as a condition is true. The syntax is as follows:

while condition:
    code block

Example:

Copy code
i = 0
while i < 5:
    print(i)
    i += 1

1

3

The for loop:

The for loop iterates over a sequence (such as a list or string) and executes a block of code for each item in the sequence. The syntax is as follows:

for item in sequence:
    code block

Example:

animals = ['cat', 'dog', 'bird']
for animal in animals:
    print(animal)

Output:

cat
dog
bird

The range() function:

The range() function generates a sequence of numbers that can be used in a loop. The syntax is as follows:

range(start, stop, step)

Example:

for i in range(1, 6, 2):
    print(i)

Output:

Copy code
1
3
5

The break statement:

The break statement allows you to exit a loop prematurely. The syntax is as follows:

while condition:
    code block
    if condition:
        break

Example:

i = 0
while True:
    print(i)
    i += 1
    if i >= 5:
        break

Output:

0
1
2
3
4

The continue statement:

The continue statement skips the current iteration of a loop and continues with the next iteration. The syntax is as follows:

for item in sequence:
    if condition:
        continue
    code block

Example:

animals = ['cat', 'dog', 'bird']
for animal in animals:
    if animal == 'cat':
        continue
    print(animal)

Output:

dog
bird

That’s it! With these basic concepts, you can start writing loops in Python.
