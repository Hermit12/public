Matlab Syntax

Wichtige Grundbefehle


Hier ist eine Zusammenstellung von Grundbefehlen zum schnellen Nachschlagen für das Matrix Laboratory, oder kurz Matlab. Ein Matlab File mit weiteren wichtigen Grundbefehlen gibt es hier. Man kann es einfach in eine neue Datei hineinkopieren und die Befehle mittels ‘Step run’ nacheinander durchlaufen lassen, um zu sehen, was sie bewirken.


Befehl Beschreibung
pwd Print Working Directory
cd change directory, z.B. cd(mypath) ändert das Verzeichnis zu einer vorher definierten Variable ‘mypath’.
clc Löscht den Inhalt des Command Windows
ls List Objects - listet die Dateien im aktuellen Verzeichnis auf
edit Öffnet die angehängte Datei, z.B. edit startup öffnet die Datei ‘startup.m’ im aktuellen Verzeichnis.
userpath Gibt den userpath aus
who Listet aktuelle Objekte im Workspace auf
whos Mehr Details zu aktuellen Objekten im Workspace
clear Löscht die Memory/Objekte im Workspace
which Gibt an, ob die nachfolgende Variable vergeben ist. Z.B. which xfür Variable ‘x’.
clear i Löscht die Variable ‘x’
eps Kleinst mögliche Nummer. 1 + eps > 1
== Vergleicht zwei Variablen miteinander und gibt logisch true (1) oder false (0) aus. f=(2==3)gibt 0 aus.
xr=[1 3 5] Nummerischer Reihen-Vektor mit 3 Zeilen. Vektoreintagungen können mit einem Abstand oder Beistrich getrennt sein.
' Transponiert einen Vektor. xc=xr' macht aus dem Reihenvektor xr den Spaltenvektor xc.
xl=[1 3 5;4 5 6] Strichpunkt erstellt eine weitere Reihe in der Matrix.
unique() Gibt die Eigenwerte eines Vektors aus.
clear Löscht die Memory/Objekte im Workspace
